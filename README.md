# PDF工具箱

纯前端 PDF 处理工具，支持合并、拆分、批量添加文字/图片水印。所有文件处理都在浏览器本地完成，不会上传到服务器。

## 核心功能

### 合并 PDF
- 支持一次上传多个 PDF。
- 按上传顺序合并。
- 显示文件大小和页数。

### 拆分 PDF
- 支持页面范围输入，例如：`1-3, 5, 7-9`。
- 自动显示 PDF 总页数。
- 下载提取后的新 PDF。

### 批量添加水印
- 多文件批量处理，输出 ZIP。
- 支持多个水印配置叠加。
- 可为每个文件单独选择要应用的水印。
- 支持复制/删除水印配置。
- 支持紧凑布局，桌面端尽量在一页完成配置。

#### 文字水印
- 支持中文和英文。
- 可设置文字、字号、颜色、透明度、旋转角度、缩放。
- 支持九宫格位置、自定义百分比坐标。
- 支持单次、固定重复、铺满三种重复模式。
- 支持全部页面或自定义页面。

#### 图片水印
- 支持 PNG、JPG/JPEG。
- 支持图片预览。
- 可设置透明度、旋转角度、缩放、位置、重复模式和页面范围。

## 关于受保护 PDF

本工具仅用于处理你有权访问和编辑的 PDF。当前版本支持未加密 PDF，以及部分无需用户密码即可打开的 PDF。若文件需要密码，请先使用你有权限的工具解除密码保护后再处理。

## 使用方法

1. 在浏览器中打开 `index.html`。
2. 从左侧选择“合并PDF / 拆分PDF / 添加水印”。
3. 上传 PDF 文件。
4. 配置参数。
5. 点击处理按钮并下载结果。

也可以使用本地服务器：

```bash
python -m http.server 8010
```

然后访问：

```text
http://localhost:8010
```

> 如果 8000 端口已被占用，请改用 8010 或其他空闲端口。

## 技术栈

- HTML5 / CSS3
- JavaScript ES6+
- PDF-Lib
- JSZip
- PDF.js（用于基础 PDF 支持）

## 浏览器兼容性

- Chrome / Edge 90+
- Firefox 88+
- Safari 14+

## 注意事项

- 大文件 PDF 会占用较多浏览器内存，建议单个文件小于 50MB。
- 图片水印建议使用透明 PNG，以获得更好的视觉效果。
- 批量处理大量文件时请耐心等待。
- 输出文件不会覆盖原文件，水印批量处理会打包为 ZIP 下载。
